# EC2_Linuxログイン手順

AWS Systems Manager セッションマネージャーを利用し、EC2_Linuxへログインする手順です。

## 前提条件

- 必要なIAM権限（`AmazonSSMManagedInstanceCore` など）が付与されている
- ログイン対象EC2インスタンスにSSMエージェントがインストール・起動済み
- EC2インスタンスがインターネットまたはVPCエンドポイント経由でSSMにアクセス可能

## ログイン手順（すべてAWS画面で実施）

1. AWSマネジメントコンソールにログイン
2. Systems Manager サービス画面へ移動
    サービス検索で「Systems Manager」を選択します。
3. 左メニューから「セッションマネージャー」を選択
4. 「セッションの開始」ボタンをクリック
5. 対象のEC2インスタンスを選択し「セッションの開始」
    インスタンス一覧からログインしたいEC2を選択し、「セッションの開始」をクリックします。
6. ブラウザ上でEC2に接続される
