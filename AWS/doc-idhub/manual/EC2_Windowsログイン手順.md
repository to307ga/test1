# EC2_Windowsログイン手順

AWS Systems Manager フリートマネージャーを利用し、EC2_Windowsへログインする手順です。

## 前提条件

- 必要なIAM権限（`AmazonSSMManagedInstanceCore` など）が付与されている
- ログイン対象Windows EC2インスタンスにSSMエージェントがインストール・起動済み
- EC2インスタンスがインターネットまたはVPCエンドポイント経由でSSMにアクセス可能

## ログイン手順（すべてAWS画面で実施）

1. AWSマネジメントコンソールにログイン
2. Systems Manager サービス画面へ移動
   - サービス検索で「Systems Manager」を選択します。
3. 左メニューから「フリートマネージャー」を選択
4. インスタンス一覧から対象EC2を選択
5. 「ノードアクション」> 「接続」> 「リモートデスクトップで接続」をクリック
6. ログイン情報を設定・入力して「接続」をクリック
    - PoC環境はパスワード認証
7. ブラウザ上でEC2に接続される
