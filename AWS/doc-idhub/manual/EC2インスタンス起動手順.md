# EC2インスタンスの起動手順

AWSマネジメントコンソール（Web画面）を利用して、停止中のEC2インスタンスを起動する手順を記載します。

## 前提条件

- AWSアカウントにログインできること
- EC2インスタンスの起動権限（StartInstances）が付与されていること

## 手順

1. AWSマネジメントコンソールにログイン
2. EC2 サービス画面へ移動
   - サービス検索で「EC2」を選択します。
3. インスタンス一覧の表示
   - 左メニューから「インスタンス」を選択します。
4. インスタンスの起動
   - 対象のインスタンスを選択＞[インスタンスの状態]＞[インスタンスの開始]をクリック。
5. 起動状態の確認
   - インスタンスの状態が[実行中]に更新されること
   - ステータスチェックが[3/3 のチェックに合格しました]に更新されること
